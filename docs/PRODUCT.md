# Product

## What
Quiz Generator ist eine Web-App für Lehrer, die per KI Multiple-Choice-Tests generiert, veröffentlicht und die Ergebnisse ihrer Schüler verwaltet. Schüler treten über einen Code oder Link bei und absolvieren Tests online.

## Who
- **Lehrer** — erstellen, bearbeiten, veröffentlichen und verwalten Tests
- **Schüler** — treten Tests bei und sehen nach Abgabe ihre Ergebnisse
- **Admin (Philipp)** — schaltet Lehrer frei, verwaltet globale Einstellungen

## Features
- KI-gestützte Quiz-Generierung (Thema, Schwierigkeit, Zielgruppe, Fragenanzahl)
- Single Choice und Multiple Choice pro Frage konfigurierbar
- Anzahl Antwortoptionen anpassbar (Default 4)
- Quiz-Editor: Fragen/Antworten nachträglich bearbeiten, löschen, hinzufügen
- Entwürfe speichern (Draft-Status)
- Veröffentlichen mit Zugangscode + Beitrittslink
- Live-Status bei aktiven Tests (Teilnehmer gesamt/fertig/offen, 30s Polling)
- Test schließen, archivieren, löschen
- Ähnlichen Test aus Vorlage generieren
- Test mit anderen Lehrern teilen (Share-Link)
- Ergebnis-Export
- Schüler: Keyboard-Shortcuts (1-4 Auswahl, Enter bei Multiple Choice)
- Schüler: Ergebnisse nach Abgabe einsehen
- Lehrer-Auth via E-Mail + Passwort (Supabase Auth)
- Admin-Freischaltung für neue Lehrer
